# Phase 7: UI Upgrade via Stitch AI

## Goal
Integrate premium, modern, and responsive UI/UX designs generated by Stitch AI into the existing React frontend, utilizing TailwindCSS and a glassmorphism aesthetic.

## Scope
1. **Design System Integration**
   - Install `tailwindcss` v3 in the Vite environment.
   - Extract Stitch AI's custom color tokens, fonts (Plus Jakarta Sans, Material Symbols), and configurations into `tailwind.config.js`.
   - Implement dynamic light/dark mode by mapping theme tokens to CSS Variables inside `index.css`.

2. **Component Refactoring**
   - **App Layout**: Introduce a new responsive shell featuring a Sidebar (`Sidebar.jsx`), Top Navigation Bar, and scrollable content canvas.
   - **DashboardSummary**: Upgrade to utilize `.glass-card` elements with progress bars and advanced Recharts configurations.
   - **TransactionList**: Convert simple lists into a professional HTML table format with conditional badging and hover effects.
   - **TransactionForm**: Transform the inline form into a centered, animated Modal overlay.

## Acceptance Criteria
- [x] Tailwind CSS is fully integrated and functional.
- [x] The UI strictly follows the Stitch AI HTML layouts and design tokens.
- [x] Application successfully supports switching between light and dark themes using the newly mapped CSS variables.
- [x] Frontend runs without logic errors (CRUD operations intact) while utilizing the new component structures.

## Execution Steps
- Retrieve Stitch AI design HTML and JSON configurations.
- Execute Tailwind installation `npm install -D tailwindcss postcss autoprefixer`.
- Map the parsed JSON color tokens to CSS variables in `index.css`.
- Systematically rewrite `App.jsx`, `DashboardSummary.jsx`, `TransactionList.jsx`, and `TransactionForm.jsx` to map React state hooks to the Stitch UI elements.

## Verification
- Run local development server (`npm run dev`).
- Utilize automated browser testing (or manual QA) to visually confirm the dashboard, modal interactions, and table rendering accurately reflect the Stitch mockups.
