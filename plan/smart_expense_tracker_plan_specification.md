# PROJECT SPECIFICATION: SMART EXPENSE TRACKER

## 1. Project Overview
A web-based Smart Expense Tracker built over 7 days. The core focus is strictly adhering to Context Engineering principles for highly consistent, high-quality code generation.
- **Frontend:** React
- **Database/Environment:** Laragon (MySQL)
- **Core Features (MVP):** Overview Dashboard, Transaction Management (CRUD, Filter), Modern UI/UX with Dark/Light Mode toggle.
